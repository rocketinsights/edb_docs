The `ALL_OBJECTS` view provides information about all objects that
reside in the database.

<div class="tabularcolumns">

Y{0.3}

</div>

|              |                   |                                                                                                                                           |
| ------------ | ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Name         | Type              | Description                                                                                                                               |
| owner        | TEXT              | User name of the object’s owner.                                                                                                          |
| schema\_name | TEXT              | Name of the schema in which the object belongs.                                                                                           |
| object\_name | TEXT              | Name of the object.                                                                                                                       |
| object\_type | TEXT              | Type of the object – possible values are: INDEX, FUNCTION, PACKAGE, PACKAGE BODY, PROCEDURE, SEQUENCE, SYNONYM, TABLE, TRIGGER, and VIEW. |
| status       | CHARACTER VARYING | Whether or not the state of the object is valid. Currently, Included for compatibility only; always set to VALID.                         |
| temporary    | TEXT              | Y if a temporary object; N if this is a permanent object.                                                                                 |
